# 📝 Notex – Smart Notes, Simplified

Notex is a modern, fast, and intuitive **note-taking web application** designed to help users capture ideas, organize notes, and access them anytime with a clean and seamless user experience.

---

## 🚀 Features

- Create, edit, and delete notes
- Instant search and filtering
- Clean and organized UI
- Secure authentication
- Cloud-based data storage
- Fast and responsive performance
- Fully responsive (mobile & desktop)

---

## 🛠️ Tech Stack

### Frontend

- React.js
- TypeScript
- Tailwind CSS
- Redux / Context API
- Vite

### Backend

- Appwrite

### Database

- Appwrite

---

## 🧠 Why Notex?

- Minimal yet powerful note management
- Optimized for speed and usability
- Scalable and clean architecture
- Built with modern full-stack best practices
